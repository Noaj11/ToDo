# 🚀 Laravel ToDo App

A modern, responsive ToDo application built with **Laravel 10** and **Bootstrap 5**.

## ✨ Features

- ✅ **Full CRUD Operations**
- ✅ **Toggle Complete/Incomplete**
- ✅ **Priority Levels** (Low/Medium/High)
- ✅ **Pagination**
- ✅ **Form Validation**
- ✅ **Responsive Design**
- ✅ **Flash Messages**
- ✅ **Confirmation Dialogs**
